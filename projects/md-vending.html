M&D Vending

Business website for a premium vending machine operation targeting Rhode Island nightlife venues.

HTML / CSS / JS GitHub Pages Business
Visit Live Site →

Overview

M&D Vending places premium vending machines inside bars, clubs, and nightlife venues across Rhode Island. The business operates on a 15% profit-sharing model — venues get a cut with zero upfront cost, making it an easy sell.

The website was built to serve as the primary lead generation tool for acquiring new venue partnerships. It needed to communicate the value proposition fast and make it dead simple for venue owners to get in touch.

Key Features

  • Clean, mobile-first landing page optimized for quick conversions
  • Clear breakdown of the profit-sharing model
  • Contact form for venue owners to request a machine
  • Responsive design that works on any device

What I Learned

Building this reinforced how to think about a website as a business tool — not just a design exercise. I worked through the full GitHub Pages deployment and custom domain setup process, and learned to make every section decision around one goal: getting a venue owner to reach out.
