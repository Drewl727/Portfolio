Sports Value Bet Finder

Automated tool that identifies value bets across Rhode Island-legal sportsbooks using live odds data.

Python The Odds API Excel

Overview

The Value Bet Finder pulls live odds from The Odds API, compares lines across multiple RI-legal sportsbooks, and flags bets where the implied probability is lower than the estimated true probability. It's built to give a statistical edge rather than relying on gut picks.

How It Works

  • Pulls real-time odds data from The Odds API across multiple sportsbooks
  • Calculates implied probabilities and identifies discrepancies between books
  • Filters for RI-legal platforms only
  • Exports picks and analysis to Excel for tracking performance over time
  • Supports multiple sports and bet types

What I Learned

This project pushed me into working with live APIs, handling rate limits, and structuring data pipelines that feed into Excel. The biggest challenge was defining "value" in a way that's statistically meaningful ‚Äî not just finding line differences, but quantifying edge. I also built out the Excel tracking sheet to log results and measure ROI over time.
